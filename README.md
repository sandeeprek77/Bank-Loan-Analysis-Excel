# Bank Loan Analysis Dashboard 📊

## 📌 Project Overview
This project focuses on analyzing bank loan data to understand lending trends, borrower behavior, and loan performance Excel.

## 🎯 Objectives
- Analyze loan applications and repayments
- Identify good vs bad loans
- Understand borrower behavior
- Visualize key business insights

## 🛠️ Tools Used
- Excel
- Pivot Tables & Pivot Charts
- Excel Data Cleaning
- Slicers for dynamic insights
- Lookup functions & calculated fields
- Dashboard structuring & design
- KPI cards & professional dashboard design

## 📊 Key Insights
- Majority of loans are Good Loans
- Debt consolidation is the most common purpose
- Most loans are issued for 36 months
- Certain regions have higher loan demand

## 📁 Files Included
- Excel Dashboard (xlsx)
- Problem Statement
- Dashboard Screenshots pdf

## 🚀 Conclusion
This project helped in understanding how banking data is used to make business decisions and analyze risk in the banking sector.
